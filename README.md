# Dragonborn Pyromancer

A fast-paced top-down sci-fi survival shooter with escalating enemy pressure, evolving boss fights, and responsive controls for desktop and mobile.

## Play

- Game link: https://milak-web.github.io/dragonborn-pyromancer/

## Highlights

- Progressive wave escalation with stronger enemy behaviors over time.
- Multi-phase bosses with unique attack patterns.
- Touch controls support (virtual joystick + action buttons) for phones/tablets.
- Adaptive screen and HUD behavior for different resolutions.
- Runtime stability guards to reduce freeze/crash scenarios.

## Controls

- Keyboard: `WASD` / `Arrow Keys` move, `Space` dash, `E` surge, `Q` special weapon
- Controller: Left stick move, `A/RT` dash, `X/Y/RB` surge, `LB` special
- Mobile: On-screen joystick + `Dash`, `Surge`, `Weapon` buttons

## Project Structure

- `index_enemy_boss_escalation_rebuild.html` - main game code (rendering, systems, AI, UI)
- `assets/ships/player_core_blue.png` - current player ship sprite

## Notes

- This project is actively tuned for difficulty, stability, and content expansion.
- Additional assets (ships, enemies, bosses, VFX, UI packs) can be dropped into `assets/` for future upgrades.
